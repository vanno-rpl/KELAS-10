HTML5

HTML adalah bahasa markup standar yang digunakan untuk membuat dan menyusun konten halaman web.

CSS3

CSS adalah bahasa stylesheet yang digunakan untuk mendeskripsikan tampilan dokumen yang ditulis dalam bahasa markup seperti HTML.

JavaScript

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web interaktif dan dinamis.
